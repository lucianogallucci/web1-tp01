Investigar que hacen las etiquetas span, i, b, a

"span" sirve como contenedor de lìnea, y le da estilo y color al texto o agrupar elementos de lìnea.

"i" Muestra el texto marcado con un estilo en cursiva o italica.

"b" Indica que el texto debe ser representado con una variable bold, o negrita,de la tipografía que se esté usando.

"a" crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

En la estructura bàsica del documento HTML, ¿Còmo definirìa html- head- body?

"HTML" representa la raiz de un documento HTML. El resto de elementos descienden de este elemento.

"head" Este elemento actúa como un contenedor para todo lo que desea incluir en la página HTML, que no es el contenido que la página mostrará a los espectadores.

"body" Contiene todo el contenido que se muestra en la página.

¿Què es HTML?

"HTML" (HyperText Markup Language, por sus siglas en inglés) es un lenguaje de marcado que indica a los navegadores web cómo estructurar las páginas web que visita.
